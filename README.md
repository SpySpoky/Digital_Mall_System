# Digital Mall - E-commerce Platform

## About the Project
A multi-functional e-commerce platform with role-based access: Admin, Manager, Courier, and Customer.

## Technologies
- PHP 8.x
- MySQL / MariaDB
- Tailwind CSS
- Bootstrap 5
- JavaScript

## Features by Role

### Admin
- User management (CRUD)
- Shop management
- Category management
- Reports & analytics

### Manager
- Product management
- Order management
- Courier assignment
- Delivery management

### Courier
- View assigned deliveries
- Update delivery status
- Delivery history

### Customer
- Browse shops and products
- Shopping cart
- Order checkout
- Reviews & ratings
